Elementos Interactivos y Gráficos en HTML

Elementos Interactivos

  • <button> - Botones interactivos
  • <details> y <summary> - Contenido expandible
  • <dialog> - Ventanas modales
  • <menu> - Menús contextuales

Elementos Gráficos

  • <img> - Imágenes
  • <svg> - Gráficos vectoriales
  • <canvas> - Dibujo y gráficos dinámicos
  • <picture> - Imágenes responsivas

Ejemplo Práctico

Código:

<!-- Ejemplo de botón interactivo -->
<button onclick="alert('¡Hola!')">Haz clic aquí</button>

<!-- Ejemplo de details/summary -->
<details>
    <summary>Ver más información</summary>
    <p>Aquí hay contenido adicional que se puede expandir y contraer.</p>
</details>

<!-- Ejemplo de SVG -->
<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red"/>
</svg>

<!-- Ejemplo de imagen responsiva -->
<picture>
    <source media="(min-width: 800px)" srcset="imagen-grande.jpg">
    <source media="(min-width: 400px)" srcset="imagen-mediana.jpg">
    <img src="imagen-pequeña.jpg" alt="Imagen responsiva">
</picture>

Resultado:

Ver más información

Aquí hay contenido adicional que se puede expandir y contraer.

Consideraciones importantes

  • Los elementos interactivos deben ser accesibles mediante teclado
  • Siempre proporcionar textos alternativos para imágenes
  • Considerar el rendimiento al usar elementos gráficos pesados
  • Asegurar que los elementos interactivos sean intuitivos para el usuario
