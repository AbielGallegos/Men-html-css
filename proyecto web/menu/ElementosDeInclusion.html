Estructura básica de un documento HTML

Un documento HTML está compuesto por diferentes elementos que definen su estructura básica. Estos son los elementos principales:

  • <!DOCTYPE html> - Declara que el documento es HTML5
  • <html> - El elemento raíz que contiene todo el documento HTML
  • <head> - Contiene metadatos, título y enlaces a recursos externos
  • <body> - Contiene el contenido visible de la página

Ejemplo de estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>
    <h1>Título principal</h1>
    <p>Contenido del párrafo</p>
</body>
</html>

Explicación de los elementos:

El elemento <head>

Contiene información importante como:

  • Codificación de caracteres (charset)
  • Título de la página
  • Enlaces a hojas de estilo CSS
  • Enlaces a scripts JavaScript
  • Meta información

El elemento <body>

Contiene todos los elementos visibles como:

  • Encabezados (h1-h6)
  • Párrafos
  • Imágenes
  • Enlaces
  • Tablas
  • Formularios
  • Y más elementos HTML
