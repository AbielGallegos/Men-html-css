Meta Información y Recursos en HTML

Etiquetas Meta

Las etiquetas meta proporcionan información sobre el documento HTML. Se colocan en el elemento <head> y no son visibles para los usuarios.

Tipos comunes de meta tags:

  • charset: Define la codificación de caracteres
  • viewport: Controla la visualización en dispositivos móviles
  • description: Describe el contenido de la página
  • keywords: Palabras clave para motores de búsqueda
  • author: Especifica el autor del documento
  • robots: Instrucciones para los bots de búsqueda

Ejemplo de meta tags:

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Descripción de mi página web">
    <meta name="keywords" content="HTML, CSS, JavaScript, programación">
    <meta name="author" content="Tu Nombre">
    <meta name="robots" content="index, follow">
    
    <!-- Open Graph meta tags para redes sociales -->
    <meta property="og:title" content="Título de la página">
    <meta property="og:description" content="Descripción para redes sociales">
    <meta property="og:image" content="url-de-la-imagen.jpg">
    
    <title>Título de la página</title>
</head>

Recursos Externos

HTML permite enlazar recursos externos como CSS y JavaScript:

Enlaces a recursos externos:

<!-- CSS -->
<link rel="stylesheet" href="estilos.css">

<!-- JavaScript -->
<script src="script.js"></script>

<!-- Favicon -->
<link rel="icon" type="image/x-icon" href="favicon.ico">

<!-- Fuentes web -->
<link href="https://fonts.googleapis.com/css2?family=Open+Sans" rel="stylesheet">

Atributos importantes de meta

  • name: Especifica el tipo de meta información
  • content: Especifica el contenido/valor
  • property: Usado principalmente para Open Graph
  • http-equiv: Proporciona información del encabezado HTTP
