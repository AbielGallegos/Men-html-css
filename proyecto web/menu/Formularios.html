Formularios en HTML

Elementos básicos de formularios

Los formularios son elementos fundamentales en HTML que permiten recopilar datos del usuario.

Elementos principales:

  • <form> - Contenedor principal del formulario
  • <input> - Campo de entrada (texto, número, email, etc.)
  • <textarea> - Campo de texto multilínea
  • <select> - Lista desplegable
  • <button> - Botón para enviar o resetear
  • <label> - Etiqueta para los campos

Ejemplo de formulario:














Código del ejemplo:

<form action="#" method="post">
    <div>
        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" name="nombre" required>
    </div>

    <div>
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
    </div>

    <div>
        <label for="edad">Edad:</label>
        <input type="number" id="edad" name="edad" min="0" max="120">
    </div>

    <div>
        <label for="mensaje">Mensaje:</label>
        <textarea id="mensaje" name="mensaje" rows="4" cols="50"></textarea>
    </div>

    <div>
        <label for="pais">País:</label>
        <select id="pais" name="pais">
            <option value="españa">España</option>
            <option value="mexico">México</option>
            <option value="argentina">Argentina</option>
        </select>
    </div>

    <div>
        <button type="submit">Enviar</button>
        <button type="reset">Limpiar</button>
    </div>
</form>

Tipos de input más comunes:

  • text - Texto simple
  • password - Campo de contraseña
  • email - Correo electrónico
  • number - Números
  • date - Fecha
  • checkbox - Casillas de verificación
  • radio - Botones de opción única
  • file - Subida de archivos
