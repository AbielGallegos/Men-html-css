Imágenes y Multimedia en HTML

Imágenes

Las imágenes se insertan usando la etiqueta <img>. Los atributos principales son:

  • src: La ruta de la imagen
  • alt: Texto alternativo para accesibilidad
  • width: Ancho de la imagen
  • height: Alto de la imagen

Ejemplo de imagen:

<img src="imagen.jpg" 
     alt="Descripción de la imagen" 
     width="300" 
     height="200">

Elementos de Video

Los videos se pueden insertar usando la etiqueta <video>:

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Tu navegador no soporta el elemento de video.
</video>

Elementos de Audio

El audio se puede insertar usando la etiqueta <audio>:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Tu navegador no soporta el elemento de audio.
</audio>

Formatos soportados

Imágenes:

  • JPEG/JPG - Para fotografías
  • PNG - Para imágenes con transparencia
  • GIF - Para animaciones simples
  • WebP - Formato moderno con mejor compresión
  • SVG - Para gráficos vectoriales

Video:

  • MP4
  • WebM
  • OGG

Audio:

  • MP3
  • WAV
  • OGG
