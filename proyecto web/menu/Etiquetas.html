Etiquetas de Texto

  • <p> - Para párrafos de texto
  • <strong> - Para texto importante
  • <em> - Para texto enfatizado
  • <mark> - Para texto resaltado
  • <sub> - Para subíndices
  • <sup> - Para superíndices

Ejemplo Práctico

Código:

<article>
    <h1>El Agua (H<sub>2</sub>O)</h1>
    <p>El agua es <strong>esencial</strong> para la vida. Es una 
    sustancia que se encuentra en estado líquido a temperatura ambiente.</p>
    <p>La fórmula química del agua es H<sub>2</sub>O, lo que significa 
    que tiene <em>dos átomos de hidrógeno</em> y <em>uno de oxígeno</em>.</p>
    <p>La temperatura de ebullición es 100<sup>o</sup>C.</p>
</article>

Resultado:

El Agua (H2O)

El agua es esencial para la vida. Es una sustancia que se encuentra en estado líquido a temperatura ambiente.

La fórmula química del agua es H2O, lo que significa que tiene dos átomos de hidrógeno y uno de oxígeno.

La temperatura de ebullición es 100oC.
