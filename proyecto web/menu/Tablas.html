Tablas en HTML

Las tablas en HTML se utilizan para mostrar datos en formato de filas y columnas. Los elementos principales son:

  • <table> - Define una tabla
  • <tr> - Define una fila
  • <th> - Define una celda de encabezado
  • <td> - Define una celda de datos

Ejemplo de una tabla básica:

Nombre Edad Ciudad
Ana García 25 Madrid
Juan López 30 Barcelona

Código del ejemplo:

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
        <th>Ciudad</th>
    </tr>
    <tr>
        <td>Ana García</td>
        <td>25</td>
        <td>Madrid</td>
    </tr>
    <tr>
        <td>Juan López</td>
        <td>30</td>
        <td>Barcelona</td>
    </tr>
</table>

Elementos adicionales de tabla:

  • <thead> - Agrupa el contenido del encabezado
  • <tbody> - Agrupa el contenido del cuerpo
  • <tfoot> - Agrupa el contenido del pie
  • <caption> - Define un título para la tabla

Atributos comunes:

  • colspan - Combina columnas
  • rowspan - Combina filas
  • border - Define el borde de la tabla
  • width - Define el ancho de la tabla
