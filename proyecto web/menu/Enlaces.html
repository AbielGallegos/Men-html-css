Enlaces y Navegación

Enlaces básicos

Los enlaces se crean con la etiqueta <a> y el atributo href:

<!-- Enlace a una página externa -->
<a href="https://www.ejemplo.com">Visita nuestro sitio web</a>

<!-- Enlace a una página interna -->
<a href="contacto.html">Contacto</a>

<!-- Enlace a una sección específica -->
<a href="#seccion1">Ir a sección 1</a>

<!-- Enlace para descargar un archivo -->
<a href="documento.pdf" download>Descargar PDF</a>

<!-- Enlace que abre en una nueva pestaña -->
<a href="https://www.ejemplo.com" target="_blank">Abrir en nueva pestaña</a>

Tipos de rutas

  • Absoluta: https://www.ejemplo.com/pagina.html
  • Relativa al dominio: /carpeta/pagina.html
  • Relativa al documento: ./pagina.html o ../carpeta/pagina.html

Ejemplo práctico de navegación

Estructura de navegación:

<nav class="menu-principal">
    <ul>
        <li><a href="index.html">Inicio</a></li>
        <li><a href="productos.html">Productos</a></li>
        <li><a href="servicios.html">Servicios</a></li>
        <li><a href="contacto.html">Contacto</a></li>
    </ul>
</nav>

Atributos importantes de enlaces

  • href: URL del destino
  • target: Dónde abrir el enlace (_blank, _self)
  • rel: Relación con la página enlazada
  • download: Para descargar el recurso
