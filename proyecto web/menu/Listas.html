Listas en HTML

HTML ofrece tres tipos principales de listas para organizar el contenido:

1. Listas Ordenadas (ol)

Se utilizan cuando el orden de los elementos es importante. Cada elemento se numera automáticamente.

Ejemplo de lista ordenada:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
</ol>

2. Listas No Ordenadas (ul)

Se utilizan cuando el orden no es importante. Los elementos se marcan con viñetas.

Ejemplo de lista no ordenada:

  • Manzanas
  • Naranjas
  • Plátanos
<ul>
    <li>Manzanas</li>
    <li>Naranjas</li>
    <li>Plátanos</li>
</ul>

3. Listas de Definición (dl)

Se utilizan para crear una lista de términos y sus definiciones.

Ejemplo de lista de definición:

HTML
Lenguaje de marcado para crear páginas web
CSS
Lenguaje para estilizar páginas web
<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado para crear páginas web</dd>
    <dt>CSS</dt>
    <dd>Lenguaje para estilizar páginas web</dd>
</dl>

Listas Anidadas

También es posible anidar listas dentro de otras listas:

  • Frutas
    • Cítricos
      • Naranja
      • Limón
    • Tropicales
      • Mango
      • Piña
