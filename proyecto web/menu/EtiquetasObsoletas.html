Etiquetas HTML Obsoletas

Algunas etiquetas que se usaban en versiones anteriores de HTML ya no se recomiendan en HTML5. Estas etiquetas han sido reemplazadas por alternativas más semánticas o CSS.

Etiquetas Obsoletas Comunes:

  • <font> - Para formatear texto (usar CSS en su lugar)
  • <center> - Para centrar contenido (usar CSS text-align)
  • <strike> - Para texto tachado (usar <del> o CSS)
  • <frame> y <frameset> - Para dividir la ventana (usar CSS grid/flex)
  • <big> - Para texto grande (usar CSS font-size)
  • <tt> - Para texto monoespaciado (usar <code> o CSS)
  • <marquee> - Para texto en movimiento (usar CSS animations)

Ejemplo de código antiguo vs. moderno:

Código obsoleto:

<center>
    <font color="red" size="5">
        <b>Texto importante</b>
    </font>
    <marquee>Texto en movimiento</marquee>
</center>

Código moderno equivalente:

<div class="texto-centrado">
    <p class="texto-destacado">
        <strong>Texto importante</strong>
    </p>
    <div class="animacion-texto">Texto en movimiento</div>
</div>

/* CSS correspondiente */
.texto-centrado {
    text-align: center;
}
.texto-destacado {
    color: red;
    font-size: 1.5em;
}
.animacion-texto {
    animation: deslizar 5s linear infinite;
}

¿Por qué evitar estas etiquetas?

  • No siguen los estándares web modernos
  • Mezclan contenido con presentación
  • Pueden causar problemas de compatibilidad
  • Dificultan el mantenimiento del código
